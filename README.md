# 新思想学习社（New Thought Learning Community）
 欢迎来到新思想学习社的线上投稿平台。本社致力于组织社员深入学习马克思主义理论，通过理论研讨、征文投稿、实践分享等形式，为大家提供学习平台。
 ---
 ## 📝 投稿须知
 1.  **投稿主题**：围绕马克思主义进学术研究。
 2.  **稿件要求**：
     *   立场鲜明，逻辑清晰。
     *   字数建议在 1500–5000 字之间，特殊题材可适当调整。
     *   格式：Word 文档或纯文本均可，可配图说明。
 3.  **投稿方式**：将稿件发送至社内指定邮箱或负责人微信或是频道，由管理员审核后统一发布至本站。
 4.  **版权说明**：投稿即授权本社在本站及相关宣传渠道非商业性使用，作者保留著作权。
